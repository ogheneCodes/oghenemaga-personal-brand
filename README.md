## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/oghenemaga-personal-brand.git
```

Navigate into the project:

```bash
cd oghenemaga-personal-brand
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
